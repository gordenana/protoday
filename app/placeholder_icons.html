Required Icon Files

To complete the Lingokids prototype, add these icon files to the root directory:

Navigation Icons

  • home.png - Home navigation icon
  • play.png - Play/Games navigation icon
  • theater.png - Theater navigation icon
  • lessons.png - Lessons navigation icon
  • family.png - Family navigation icon

Top Bar Icons

  • avatar.png - User profile avatar
  • heart.png - Favorites icon
  • search.png - Search icon
  • rewards.png - Rewards icon

Note: The app will automatically fall back to emoji placeholders if these files are not found.
